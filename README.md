## General info
A language derived from TypeScript and Python for the Summer 2020 repl.it language hackathon.

## Technologies
Project is created with:
* ts-morph: latest
* ts-node: latest
* typescript: latest

## Setup
To run this project, install it locally using npm:

$ npm install

$ npm start
